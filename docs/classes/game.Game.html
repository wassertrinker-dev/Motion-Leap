    Class Game

    Die Hauptklasse, die das gesamte Spiel orchestriert. Sie verwaltet die Spiel-Schleife, den Spielzustand (Leben, Game Over), den Spieler, die Gegner, das Rendering auf der Canvas und die Kamera-Eingabe.
